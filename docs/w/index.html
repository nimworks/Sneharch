Responsive Design with Media Queries

Aim: Create a webpage that changes background color based on screen orientation and displays a message only when the screen width is less than 600px.

<!DOCTYPE html>
<html>
<head>
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>Responsive Media Query</title>
	<style>
		body {
			font-family: Arial, sans-serif;
			padding: 20px;
			text-align: center;
		}

		/* Portrait orientation */
		@media (orientation: portrait) {
			body {
				background-color: lightblue;
			}
		}

		/* Landscape orientation */
		@media (orientation: landscape) {
			body {
				background-color: lightgreen;
			}
		}

		/* Width less than 600px */
		@media (max-width: 600px) {
			.mobile-message {
				display: block;
				color: red;
				font-weight: bold;
			}
		}

		/* Hide message by default */
		.mobile-message {
			display: none;
		}
	</style>
</head>
<body>
	<h1>Welcome to Responsive Design</h1>
	<p class="mobile-message">You are viewing this on a small screen!</p>
</body>
</html>
